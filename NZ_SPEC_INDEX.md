## Structure

- All specifications are located in the `/specs/` directory.
- Each specification has a unique numeric ID.
- IDs are assigned sequentially and never reused.
- All documents follow the template defined in `.github/SPEC_TEMPLATE.md`.

## Notes

- This index is updated manually with every new specification.
- Version numbers reflect the global standard version, not individual document maturity.
